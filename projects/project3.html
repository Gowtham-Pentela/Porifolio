Road Crack Detection

This code is designed for road crack detection using Google Street View images. It includes the following functionalities:

  • Data Collection: Utilizes the Google Street View API to gather images at different locations along specified coordinates.
  • Data Processing: Extracts metadata from the downloaded images and organizes them into a structured dataset.
  • Data Transformation: Converts the collected data into a CSV file with relevant information, including location details and image filenames.
  • Image Labeling: Labels the images with crack or no-crack classes and updates the dataset accordingly.
  • Model Development: Implements a VGG16-based Convolutional Neural Network (CNN) for road crack detection.
  • Training: Trains the CNN model on the labeled dataset to learn road crack patterns.
  • Testing: Evaluates the model on a separate test set to measure its accuracy.
  • Localization Visualization: Generates visualizations with randomly generated bounding box localizations on images for evaluation.
